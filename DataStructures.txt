LocalStorageShot
Key: Shot Id
Value: JSON.stringify(Shot)

LocalStorageConfig
Key: ConfigKey - e.g. mainStatusBatteryLevel
Value: Output from gphoto2 when you call --get-Config 'configKey' - contains all config options and current value

cameraPi.allshots - array of Shot objects
cameraPi.allconfigs - array of config objects
